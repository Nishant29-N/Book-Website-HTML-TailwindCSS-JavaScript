📚 Book Website

A modern and responsive Book Website built with HTML, CSS (TailwindCSS), and JavaScript.
This project is designed as an interactive platform to showcase books, their details, and provide users with a seamless browsing experience.

✨ Features

📖 Book Listings – Display books with titles, authors, genres, and descriptions.

🔍 Search & Filter – Easily find books by name, category, or author.

📱 Responsive Design – Fully optimized for desktop, tablet, and mobile screens.

🎨 Modern UI – Clean and minimal design with TailwindCSS.

🌙 Dark/Light Mode – Smooth toggle for better accessibility (if included).

📑 Detailed Book Pages – View book details with cover images and descriptions.

🛠️ Tech Stack

HTML5 – Structure of the website.

TailwindCSS – Styling and responsive design.

JavaScript (ES6) – Interactivity and functionality.

🚀 How to Run

Clone the repository:

git clone https://github.com/your-username/book-website.git


Navigate to the project folder:

cd book-website
